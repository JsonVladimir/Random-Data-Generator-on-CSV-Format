    librería _Faker_ con opción de exportar los datos en archivo ***.csv***
## Dependencia:
    1 - Faker
## Instalación de Faker:
    Instalación de la librería con pip:
    _pip install Faker_

    Instalación de la libería con conda:
    _conda install -c main faker_
